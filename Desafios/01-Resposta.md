# Resposta Exercicio 1 :
# 1. Instale o Docker no seu computador:

[Para Windows](https://docs.docker.com/desktop/setup/install/windows-install/) 

[WSL](https://learn.microsoft.com/pt-br/windows/wsl/install) 

[Para Linux](https://docs.docker.com/desktop/setup/install/linux/)

[Para Mac](https://docs.docker.com/desktop/setup/install/mac-install/)

# 2. Executar o Docker:

Para Windows: É necessário executar o WSL primeiro
              [Configurar o WSL no Docker](https://learn.microsoft.com/pt-br/windows/wsl/tutorials/wsl-containers)
              ** Verifique se o DockerDesktop está com todas as propriedades ativadas do WSL.
              
# 3. Verifique se está instalado corretamente:

No terminal execute:

```bash
docker --version
```
