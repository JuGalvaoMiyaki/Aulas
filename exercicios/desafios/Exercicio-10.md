# Exercício 10 – Executar Comandos Dentro de um Container em Execução (Flask)

**Objetivo:**

```bash
Aprender a usar docker exec para executar comandos dentro de um contêiner que roda uma aplicação Flask.

```
**Parte 1:** 
```bash
Crie uma imagem Docker para uma aplicação Flask simples. (Use a aplicação do repositório Git)

Inicie um contêiner em segundo plano com o nome meu_flask.

Verifique se o contêiner está rodando.
```

**Parte 2:**

```bash
Use docker exec para:

Abrir um shell dentro do contêiner.

Listar os arquivos da aplicação.

Verificar a versão do Python instalada.

Criar um arquivo dentro do contêiner

Ler o conteúdo do arquivo criado

Parar o container
```
