# Exercício 7: Investigando uma Imagem - Modo Detetive

**📖 Cenário**

O time de segurança do Bentão recebeu uma imagem nginx:alpine de uma fonte não oficial e pediu para você investigar o que tem dentro dela sem executar o container (por segurança).

**Desafio: Baixe a imagem nginx:alpine e descubra:**

```bash
Qual sistema operacional é a base

Quais portas estão expostas

Qual o comando padrão executado

Quantas camadas a imagem possui

Qual o tamanho total

Importante: Use APENAS comandos do Docker, sem rodar o container.
```
